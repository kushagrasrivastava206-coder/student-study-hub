# student-study-hub
An open educational platform providing notes, formulas, revision resources, and study tools for high school students.


## Features
- Class 10 Notes
- Physics Resources
- Chemistry Resources
- Mathematics Resources
- Study Guides

## Upcoming Features
- Class 11 Notes
- Class 12 Notes
- Practice Questions
- Formula Sheets
- Study Planner

## Mission
To provide free and accessible educational resources for students.
